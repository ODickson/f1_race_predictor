# f1_race_predictor
Predict the race result for an f1 race using a variety of datapoints

## Data (from 2003)
- FIA results and statistics:
  - Car
  - Engine
- Ergast Data:
  - Schedule
  - circuit id + alt
  - Drivers
  - Qualifying fastest lap pace difference
  - Start position
  - Finish position
- Wiki
  - Driver rookie year
- Visual Crossing
  - race day weather
    - humidity
    - temperature
    - Wet Race ( 1 or 0)
    - Cloud level (clear to overcast)
